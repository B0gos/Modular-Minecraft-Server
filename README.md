# Modular-Minecraft-Server
### ! Has not been tested or modified since ~2023 !

A Minecraft server consisting of a proxy, lobby, packet translator, and a few other sub servers to host games and survial worlds. In a modular structure achieved using docker and docker compose.

Intended and designed as a small private server between friends. 

Dependancies:
Docker
Git

To start the server run:
docker-compose up -d
